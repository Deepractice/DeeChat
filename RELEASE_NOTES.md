# DeeChat v1.0.0 - 智能桌面AI助手首次发布

🎉 **DeeChat v1.0.0 正式发布！**

## ✨ 核心特性
- 🧠 **5层智能系统提示词架构** - 真正理解用户意图的AI伙伴
- 🎯 **智能意图识别系统** - 14种对话类型精准分类
- 👤 **用户自主权原则** - 始终以用户为决策核心
- 🔧 **MCP工具生态集成** - 无缝扩展AI能力
- 🚀 **PromptX认知框架** - 专业角色动态切换

## 🏗️ 技术亮点
- **前端**: React + TypeScript + Ant Design
- **桌面**: Electron跨平台应用
- **数据**: SQLite本地数据库
- **AI**: LangChain智能对话集成
- **架构**: 模块化微服务设计

- 🍎 **macOS**: 11.0+ (Intel & Apple Silicon)

## 🚀 安装指南

### 方式1：DMG标准安装 (推荐)
1. 下载 **DeeChat-1.0.0.dmg**
2. 双击打开DMG文件
3. 将DeeChat图标拖拽到Applications文件夹
4. 从启动台或Applications启动DeeChat

### 方式2：ZIP便携安装
1. 下载 **DeeChat-1.0.0-mac.zip**
2. 解压缩得到DeeChat.app
3. 移动到Applications文件夹 (推荐)
4. 双击启动应用程序

## 🔧 系统要求
- macOS 11.0 (Big Sur) 或更高版本
- 至少 100MB 可用磁盘空间
- 网络连接（用于AI功能）
